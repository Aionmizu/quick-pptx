# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct. By participating (issues, PRs, discussions), you agree to abide by its terms.

## In short

- Be kind. Disagree on the work, not the person.
- Critique is welcome — adversarial reviews of design and code are part of how this project moves forward.
- Harassment, personal attacks, and discrimination are not.

## Reporting

If you witness or experience unacceptable behavior, contact the project owner via a [private security advisory](https://github.com/Aionmizu/quick-pptx/security/advisories/new). Reports are confidential.
